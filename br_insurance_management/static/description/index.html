Overview

The user can construct insurance plans using this module. These policies allow for the creation of insurances. We have the ability to handle insurance claims and agent salaries with or without commission.

Features

For customers, create insurance plans.

Organize and manage insurance data.

Control the agents' salaries and commissions.

Make entries in the books for everyone.

Work Methods

Insurance-> Configuration -> Customers

Configuration -> Policies. Create new policies and add the policy details.

Agent-> Agent. Create a new agent and enter the information. Fixed, commission-based, or both (salary and commission) are the available salary types. The salary payment can be made in cases when the compensation is determined by the chosen salary type. Based on the proportion specified in the insurances under the agent, the commission is determined.

Insurance Management-> Insurance . Create insurance for clients in accordance with a policy. Add the agent and the percentage of the agent's commission. After completing all necessary information, the insurance gets verified. It is possible to construct an insurance invoice.

Insurance Management-> Claims .The claim should be created using the insurance number that is generated after creating the insurance policy.
